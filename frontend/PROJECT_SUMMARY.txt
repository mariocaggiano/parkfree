================================================================================
                        CONFIGURATION REQUIRED
================================================================================

Before running the app, you must configure:

1. Firebase (src/services/auth.ts)
   - Replace with your Firebase project credentials
   - Enable auth methods: Email, Google, Apple

2. Mapbox (src/components/Map.tsx)
   - Replace MAPBOX_TOKEN with your Mapbox token
   - Get token from: https://www.mapbox.com/

3. Environment Variables (.env.local)
   Create file with:
   VITE_API_URL=http://localhost:8000/api
   VITE_MAPBOX_TOKEN=your_token
   VITE_FIREBASE_API_KEY=your_key
   VITE_FIREBASE_AUTH_DOMAIN=your_domain
   VITE_FIREBASE_PROJECT_ID=your_project
   VITE_FIREBASE_STORAGE_BUCKET=your_bucket
   VITE_FIREBASE_MESSAGING_SENDER_ID=your_sender
   VITE_FIREBASE_APP_ID=your_app_id

4. Backend API
   - Ensure backend is running on configured API_URL
   - All endpoints from COMPONENTS.md must be implemented
